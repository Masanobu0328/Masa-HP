## 🎨 デザイン要件（確定案: Refined Minimal Natural）

### デザインコンセプト: "Refined Minimal Natural"
> 「余白の美」と「個の存在感」。
> 飾り立てるのではなく、素材（Masa本人とコンテンツ）の良さを引き立てる器のようなデザイン。
> 前回のシンプル案をベースに、微細な質感とタイポグラフィで「退屈」ではなく「洗練」に見せる。

### カラースキーム
| 用途 | カラー名 | コード | 意図 |
|------|----------|--------|------|
| **Base** | オフホワイト | `#FAFAF9` | 真っ白すぎない、目に優しい紙のような質感 |
| **Text** | インクブラック | `#333333` | 視認性が高く、かつ強すぎない黒 |
| **Accent** | オリーブセージ | `#7C9082` | 自然を感じさせる落ち着いた緑。彩度を抑えてシックに。 |
| **Sub** | サンドベージュ | `#EBE8E4` | 背景の切り替えやカードの枠線に使う微細な色 |

### デザインスタイル: "Quiet Impact"
- **Focal Point (視線の集中)**
  - 顔写真（プロフィール）を円形またはソフトな角丸で配置し、ページの中心的な存在にする。
  - 写真の周囲に極めて薄い枠線や、柔らかい影を落とし、平面のっぺり感を防ぐ。

- **Typography with Character**
  - 基本はシンプルだが、名前（Masa）や見出しにだけ、少し特徴のあるセリフ体（例: Playfair Display）を使い、エレガントな「フック」を作る。
  - 本文は読みやすいサンセリフ体（Inter / JPゴシック）ですっきりと。

- **Subtle Texture**
  - 背景はベタ塗りではなく、視認できないレベルの微細なノイズを加えるか、セクションの境界を柔らかな曲線で区切り、無機質さを消す。

### ビジュアル要素
| 要素 | 仕様 |
|------|------|
| **Heroセクション** | 中央にプロフィール写真、その下に名前と肩書き。余白を贅沢に使う。 |
| **写真フレーム** | 写真を引き立てるダブルボーダー、または非対称な有機的シェイプ。 |
| **アイコン** | 線が細く、繊細な植物や幾何学モチーフ。主張しすぎない。 |
| **ボタン** | 塗りつぶしではなく、アウトラインまたは控えめな背景色。ホバーでゆっくり色が満ちる。 |

### 目指す印象
- × 地味、手抜き、寂しい
- ○ 清潔、誠実、余裕がある、本質的

### タイポグラフィ
- **見出し**: Noto Sans JP / Inter
- **本文**: Noto Sans JP（日本語）/ Inter（英語）
- **アクセント**: 手書き風フォント（強調ポイントに）

### レスポンシブ対応
- **Desktop**: 1200px以上
- **Tablet**: 768px - 1199px
- **Mobile**: 767px以下
- **完全モバイルファースト設計**
